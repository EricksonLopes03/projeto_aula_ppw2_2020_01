Sobre nosso sistema

Nosso Ecommerce foi desenvolvido com o intúito de trabalhar o conteúdo da disciplina de Programação para Web 2 na prática.

Para a confecção do trabalho foram utilizadas as seguintes tecnologias:

  • Lingagem de Programação - Java 8
  • WEB - JavaServer Faces - JSF 2.3
  • HTML 5
  • CSS 3
  • Javascript
  • Bootstrap 4
  • Primefaces 7.0
  • Tomcat 8.053
  • Java CDI - Weld 2.4.8
  • JPA - Hibernate 5.4.14
  • Mysql 8.0.20
  • FlywayDB 6.5.0
  • Lombok 1.18.12
